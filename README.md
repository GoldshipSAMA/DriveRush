# DriveRush Monorepo

- `frontend/`：Chrome 扩展前端（本地同步 + 登录态 + 云端同步按钮）
- `backend/`：Java Spring Boot 云端后端（认证 + 入库同步）

## 快速说明

### 前端

在 Chrome 开发者模式加载 `frontend/` 目录。

### 后端

进入 `backend/`，按 [backend/README.md](D:/Repository/DriveRush/backend/README.md) 启动。

默认后端地址为 `http://localhost:3000/api`，已和前端对齐。
